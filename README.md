# OMi CLi 2.0 - Modern Omi command-line interface

## Quick Start

### install

```shell
npm install -g @yuqing521/omi-cli

```

### use
```json
"bin": {
    "omi-test": "./index.js",
    "omi-create-test": "./index.js"
  },
```
**no option**

```shell
omi-test
```

**with option**

```shell
omi-create-test [project name] -t omi-admin
```

### configure option

workdir: [project name]
Template: -template / -t
